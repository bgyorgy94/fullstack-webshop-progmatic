# express-backend


### használata:

- **telepítés:**
    ```shell
    $ npm install
    ```
- **server elindítása:**
    ```shell
    $ npm start
    ```
### port megváltoztatása:

- hozz létre egy `.env` file-t a project gyökerében a `.env.example` mintájára.
- A `.env` fileban  a `PORT=3030` át lehet írni tetszőleges port számra.
